Dưới đây là bảng liệt kê các mã trạng thái HTTP phổ biến và ý nghĩa của chúng: | **Mã trạng thái** | **Tên mã** | **Mô tả ngắn gọn** | | ----------------- | --------------------- | --------------------------------------------------- | | 200 | OK | Yêu cầu thành công, server đã trả về dữ liệu. | | 201 | Created | Tài nguyên đã được tạo thành công. | | 204 | No Content | Thành công nhưng không có nội dung trả về. | | 400 | Bad Request | Yêu cầu không hợp lệ từ phía client. | | 401 | Unauthorized | Chưa xác thực hoặc thông tin xác thực không hợp lệ. | | 403 | Forbidden | Bị từ chối truy cập, không có quyền thực hiện. | | 404 | Not Found | Không tìm thấy tài nguyên yêu cầu. | | 405 | Method Not Allowed | Phương thức HTTP không được hỗ trợ. | | 500 | Internal Server Error | Lỗi máy chủ nội bộ, server gặp sự cố khi xử lý. | | 503 | Service Unavailable | Dịch vụ tạm thời không khả dụng, thường do quá tải. |
